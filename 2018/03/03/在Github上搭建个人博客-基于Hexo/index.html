将网页发布到Github上

  • 复制github博客仓库链接

  • 修改blog文件夹下的配置文件_config.yml

  • 在Git Bash上输入npm install hexo-deployer-git --save

  • github配置SSH Key。
  • 上输入hexo d，若出现INFO Deploy done: git则为成功。
  • 在浏览器上输入网址查看。

发博文

  • 方法1：在blog\source_posts文件夹下新建first blog.md文件即可创建
  • 方法2：在git bash 中输入hexo new "first blog"或hexo new post "first blog"即可创建

注意：文章是用markdown语法(待完善…)写的， 写完后，输入hexo d -g部署到github上
