Hexo的NexT个性化主题配置

发表于 2018-03-03 | 分类于 Github ， Pages |
字数统计: 677 | 阅读时长 ≈ 3

摘要： 本文介绍了如何打造个性化博客页面，包括在博文中添加图片以及在末端添加评论。其他常见设置可阅读底部的其他参考。

博文添加图片

  1. 把主页配置文件_config.yml 里的post_asset_folder:改为true

  2. 在博客根目录下执行npm install hexo-asset-image --save

  3. 完成后，运行hexo new "博文标题" 来生成博文时，/source/_posts文件夹内除了博文标题.md文件还有一个博文标题的文件夹，将图片放在该文件夹，并在文中引用即可。
